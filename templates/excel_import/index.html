Como Usar
Tipos de Importação:
  • Cronograma: Importa propostas, empresas, CNPJ, consultores, datas
  • Produtividade: Importa horas trabalhadas, produtividade por consultor
  • Considerações: Importa resumo de programas, etapas, tipos
  • Faturamento: Importa valores, datas de faturamento, status
Formatos Esperados:

As planilhas devem ter as abas corretas:
• CONSULTA INFO. para propostas
• PRODUTIVIDADE para produtividade
• RESUMO para considerações
• FATURAMENTO para faturamento

Atenção

• Dados duplicados serão atualizados
• Novos dados serão inseridos
• Verifique o formato antes de importar
• Faça backup antes de grandes importações
